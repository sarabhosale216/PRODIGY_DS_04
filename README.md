# PRODIGY_DS_04 - Sentiment Analysis on Social Media Data

## 📌 Task Objective

Analyze and visualize sentiment patterns in social media data to understand public opinion and attitudes towards different brands and topics. This project uses the Twitter Entity Sentiment Analysis dataset to explore sentiment distribution and identify trends through data visualization.

---

## 📂 Dataset

**Twitter Entity Sentiment Analysis Dataset**

The dataset contains tweets related to various brands and topics with predefined sentiment labels.

### Features

* **ID** – Unique identifier for each tweet
* **Entity** – Brand or topic mentioned in the tweet
* **Sentiment** – Sentiment label (Positive, Negative, Neutral, Irrelevant)
* **Tweet** – Tweet text

The following analyses and visualizations were performed:

* Data Cleaning

* Top 10 Most Mentioned Brands
* Brand-wise Sentiment Comparison
* Positive Tweet Word Cloud
* Negative Tweet Word Cloud
* Overall Sentiment Pie Chart

---

## 📈 Key Insights

* Positive tweets constitute a significant portion of the dataset.
* Negative sentiments reveal customer complaints and dissatisfaction.
* Neutral tweets mainly represent informational or descriptive content.
* Some brands receive substantially higher engagement than others.
* Word clouds highlight the most frequently occurring words in positive and negative tweets.


4. Open the Jupyter Notebook or Google Colab notebook.
